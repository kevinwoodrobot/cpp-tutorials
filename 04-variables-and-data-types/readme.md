# data size ranges 
https://learn.microsoft.com/en-us/cpp/cpp/data-type-ranges?view=msvc-170


1 byte = 8 bits = 2^8 = 256

4 bytes = 32 bits = 2^32 = 4,294,967,296

So int is between 
-2,147,483,648 to 2,147,483,647
(2^32 divided between pos and neg numbers)
